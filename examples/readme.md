# Example Gallery

Filare example inputs are organized by scenario:

- `basic/` — single harness diagram for quick smoke checks.
- `minimal-document/` — DIN A4 document with two harnesses and BOM only.
- `document-cut/` — three-harness document with cut diagrams enabled.
- `document-termination/` — three-harness document with termination diagrams enabled.
- `multi-page/` — multi-page harness collection with split BOM/title pages.
- `all-document/` — two complex harnesses with every output enabled.
